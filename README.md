# Indian Stock Portfolio Tracker

## Overview

This project is a simple Indian Stock Portfolio Tracker developed using Python for the CodeAlpha Python Programming Internship.

## Features

* Displays Indian stock prices
* Accepts stock name and quantity
* Calculates investment value
* Saves results to a text file
* Uses Python dictionaries and file handling

## Available Stocks

* RELIANCE
* TCS
* INFY
* HDFCBANK
* ICICIBANK
* SBIN
* ITC
* WIPRO

## How to Run

```bash
python stock_tracker.py
```
